## English

X Layer Yield Charter is live.

The rule is simple:

Agents can spend yield.
Agents cannot touch principal.

A human sets the principal floor, releases harvested yield as operating budget, and every agent spend is checked before execution with proof and receipts.

This is not another bot. It is treasury governance infrastructure for agentic DeFi on X Layer.
