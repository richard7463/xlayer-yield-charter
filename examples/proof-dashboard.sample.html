0. What This Page Does

A control surface for agent spending

X Layer Yield Charter is not a yield dashboard. It is a governance layer for autonomous agents. The owner releases a small yield budget, the agent asks to spend it, the guard checks the request, and only then can execution reach X Layer.

1. Protect

Owner locks the treasury

$4.00 is marked as principal. The agent cannot spend below this floor.

2. Release

Owner releases yield budget

$3.00 is released as agent operating budget. This is the only money the agent can use.

3. Request

Agent asks to act

yield-ops-agent requested $3.23 for OKB -> USDT0.

4. Decide

Guard resizes or blocks

The guard returned Resize and allowed $1.00.

5. Receipt

Every action leaves proof

Broadcasted on X Layer with receipt 0x15cca235...2aa321d0.

4. Operator Controls

Human owner can stop, review, or resume the agent

This is the human control layer. The agent is autonomous only while the lease is active, the route is allowed, and the released yield budget is still available.

Pause agent Stop all new spending

Used when the owner sees strange behavior or wants to freeze the agent before the next request.

Review mode Route risky requests to human review

Used when the agent can keep reporting, but cannot execute without a human checkpoint.

Resume Reopen the budget lane

Current mode is Active. The agent can operate only inside the active lease.
